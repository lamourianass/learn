Every web page shoud have a page title to describe the meaning of the page. The element adds a title to your page: <!DOCTYPE html> <html> <head> <title>HTML page Content The title is shown in the browser' title bar The title should describe the content and the meaning of the page. The page title i very important for search engin optimization (SEO). The text i used by search engine algorithm to decide the order when listing page in search results. The element: * Define a title in the browser toolbar * Provides a title for the page when it is added to favorites * Displays a title for the page in search engine-results So, try to make the title as accurate and meaningful as possible!
