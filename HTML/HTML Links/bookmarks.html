HTML links can be used to create bookmarks, so that readers can jump to specific parts of a web page. Bookmarks can be useful if a webpage is very long. To create a bookmark - first create the bookmark, then add a link to it. When the link is clicked, the page will scroll down or up to the location with the bookmark.
