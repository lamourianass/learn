Absolute URL - links to an external image that is hosted on another website. 2. Relative URL - links to an image that is hosted within the website. Here, the URL does not include the domain name. If the URL begins with a slash, it will be relative to the current page. Example: src="img_girl.jpg". If the URL beginswith a slash, it will be relative to the domain. Example: src="/images/img_girl.jpg". Tip: It is almost always best to use relative URLs. They will not break if you changed domain.
