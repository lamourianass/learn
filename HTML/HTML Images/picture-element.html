The HTML element allows you to display different pictures for different devices or screen sizes. The HTML Element gives web developers more flexibility in specifying image resources. The element countains one or more elements, each referring to differernt images through srcset attribute. This way the browser can choose the image that best fits the current view and/or device. Each element has a media attribute that defines when the image is the most suitable. Note: Always specify an element as the last child element of the element. The element is used by browsers that do not support the element, or if none of the tags match. There are two main purposes for the element: 1. Bandwidth: If you have a small screen or device, it is not necessary to load a large image file. The browser will use the first element with matching attribute values, and ignore any of the following elements. 1. Format Support: Some browsers or devices may not support all image formats. By using the element, you can add images of all formats, and the browser will use the first format it recognizes, and ignore any of the followinig elements.
